仿 Steam 游戏商城静态网站

项目简介

本项目是前端 HTML+CSS 综合实训作品集，纯原生前端技术开发仿 Steam 游戏商城多页面静态网站，无前端框架、无后端数据库。项目搭建完整多页面站点，包含首页、游戏分类页、关于页面、用户注册表单页，规范分层管理样式与图片资源，实现布局、渐变、过渡动画、表单、表格等全套前端基础视觉交互效果，可用于求职简历展示。

技术栈

·   HTML5：语义化标签、多页面跳转、表单、表格、图文混排
·   CSS3：Flex 弹性布局、浮动布局、线性渐变、圆角边框、transition 平滑过渡、hover 交互样式、样式文件拆分复用

项目目录结构
game-store-steam/
├── README.md            # 项目说明文档
├── css/
│   ├── style.css        # 全站通用样式（导航、游戏卡片、页脚、全局动画）

├── games.html           # 游戏分类列表页：侧边栏分类导航+游戏数据表格
├── about.html           # 项目介绍与功能说明页面
└── login.html           # 用户注册表单页面

核心功能实现

1.全站统一导航栏，多页面互相跳转，自动高亮当前访问页面
2.首页活动 Banner 轮播区域，展示游戏活动宣传图
3.游戏商品卡片布局：展示游戏封面、文字介绍、折扣售价、购买按钮
4.侧边栏游戏分类导航，搭配表格结构化展示游戏名称、价格、评分信息
5.完整用户注册表单：用户名、邮箱、密码、确认密码、验证码、用户协议勾选，支持提交 / 重置
6.全套 CSS 视觉效果：渐变背景、圆角、输入框焦点高亮、鼠标悬停变色、平滑过渡动画
7.统一底部版权信息，所有页面共用复用

本地运行方式

1.下载全部项目文件到本地；
2.直接双击根目录下 index.html，浏览器打开即可预览完整网站；
3.通过顶部导航栏自由切换登录、游戏列表、关于等子页面。

开发收获

1.熟练使用 HTML5 语义化标签搭建规范页面结构，掌握表单、表格、多页面网站开发思路；
2.精通 CSS3 弹性布局、浮动布局、渐变、圆角、过渡动画等核心样式技术；
3.学会工程化分层管理代码，拆分全局通用样式与页面专属样式，提升代码可维护性；
4.能够独立完成商用展示类静态前端网站，理解统一视觉规范与页面复用逻辑。
